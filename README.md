# 🥁 DrumVerse — 3D Interactive Drum Learning Experience

🌐 **Live Demo:** https://drum-verse.vercel.app/

DrumVerse is a high-performance, interactive **3D drum kit web application** designed for both **learning and free play**.  
It combines real-time audio, immersive visuals, and structured tutorials to create a **game-like music learning experience**.

---

## 🚀 Features

### 🎮 Interactive 3D Drum Kit
- Fully playable drum kit *(Kick, Snare, Hi-Hat, Tom, Crash)*
- Real-time animations *(drumsticks, pedals)*
- Mouse + keyboard interaction
- Immersive night-stage environment with dynamic lighting

---

### 🌊 Visual Resonance System
- Sound wave ripples on every hit  
- Drum surface vibration effects  
- Glow + emissive lighting feedback  
- High-impact visual timing indicators  

---

### 📘 Bollywood Learning System
- Notebook-style guided tutorials  
- Rhythm cues: **Dhum, Tak, Chak**  
- Step-by-step progression  
- Real-time sequence highlighting  

---

### 🎵 Song Mode
- Curated Bollywood playlist  
- BPM-synced learning  
- Auto-advancing tutorial flow  
- Play along with structured rhythm guidance  

---

### ⚡ Performance Optimised
- Low-latency audio *(Web Audio API)*  
- Preloaded sound assets  
- Smooth **60 FPS** animations  
- Optimised 3D rendering  

---

## 🎹 Controls

| Action        | Key     |
|--------------|--------|
| Kick         | Space  |
| Snare        | S      |
| Hi-Hat       | H      |
| Tom          | T      |
| Crash        | C      |
| Next Lesson  | Enter  |

**Additional Controls:**
- 🖱 Mouse / Touch → Play drum parts  
- 🔄 Drag → Rotate camera  
- 🔍 Scroll → Zoom  

---

## 🛠 Tech Stack

- **Frontend:** React (Vite) + TypeScript  
- **3D Engine:** Three.js via @react-three/fiber  
- **3D Tools:** @react-three/drei  
- **Animations:** Framer Motion  
- **Styling:** Tailwind CSS  
- **Audio:** Web Audio API  
- **Icons:** Lucide React
